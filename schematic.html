Quick Reference

  • I2C Bus: 400kHz
  • SPI Bus: 10MHz
  • UART: 115200 baud
  • 1-Wire: 4.7kΩ pullup resistor
  • I2C: 10kΩ pullup resistors on SDA/SCL
  • ADC: 12-bit @ 1MHz sampling
  • VCC: 3.3V for all components

Important Notes

  • Both MAX30102 sensors have the same I2C address (0x57) - requires I2C multiplexer (TCA9548A) for the second sensor
  • All I2C lines need 10kΩ pull-up resistors to 3.3V
  • 1-Wire bus needs a single 4.7kΩ pull-up resistor
  • AD8232 output is analog (0-3.3V) and connects directly to ADC pin
  • nRF52832 UART connections are crossed: STM32 TX → nRF RX, STM32 RX → nRF TX
