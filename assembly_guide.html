ASSEMBLY SEQUENCE

1 BUILD POWER SYSTEM

DO THIS FIRST! EVERYTHING NEEDS POWER.

  • Connect USB-C connector (J1) for 5V input
  • Wire BQ24075 battery charger IC with 10kΩ resistors on ISET and TS pins
  • Connect 3.7V 1000mAh LiPo battery to VBAT pin
  • Connect 20× MP3-37 solar panels in parallel configuration
  • Test voltage output with multimeter before proceeding
2 ASSEMBLE MAIN PCB
  • Solder STM32F405RGT6 microcontroller (64-pin LQFP) - use flux!
  • Add nRF52832 BLE module (pre-made module recommended)
  • Place all decoupling capacitors (100nF) near each IC
  • Solder 8MHz crystal with 22pF load capacitors
  • Add voltage regulator (AMS1117-3.3) with input/output caps
3 INSTALL SENSOR MODULES
  • AD8232 ECG: Mount module, connect 3 electrode leads
  • MAX30102 ×2: Install both PPG sensors, add TCA9548A I2C mux
  • MPU-9250: Mount IMU module on back side
  • AD5933: Install impedance analyzer with electrode connections
  • DS18B20 ×4: Wire temperature sensors to 1-Wire bus (PA8)
4 WIRE I2C AND SPI BUSES
  • Add 10kΩ pullup resistors to SDA (PA6) and SCL (PA5)
  • Connect all I2C devices to shared SDA/SCL lines
  • Wire SPI flash (W25Q128) to PB13/14/15 pins
  • Double-check all connections with schematic
CRITICAL

Both MAX30102 sensors share address 0x57, so second sensor MUST go through TCA9548A multiplexer!

5 INTEGRATE INTO GARMENT
  • Sew conductive thread pathways on compression shirt base
  • Attach ECG electrodes to chest area (RA, LA, RL positions)
  • Mount PPG sensors: one on chest, one on shoulder/wrist
  • Place temperature sensors at chest, back, and arm locations
  • Install solar panel array on back (TPU encapsulated for washability)
  • Create secure pocket for main electronics module
  • Use hot glue for strain relief on all wire junctions
6 PROGRAM & TEST
  • Flash STM32 firmware via USB using ST-Link programmer
  • Program nRF52832 with BLE stack
  • Test each sensor individually with multimeter/oscilloscope
  • Verify I2C communication at 400kHz
  • Check BLE connectivity with smartphone app
  • Run full system diagnostic and calibration

SAFETY & OPERATIONAL NOTES

  • ESD PROTECTION: Use anti-static wrist strap when handling electronics
  • SOLDERING: Work in ventilated area, use lead-free solder
  • TESTING: Never power on until all connections verified
  • WASHING: Remove electronics module before washing garment
  • BATTERY: Never puncture or short-circuit LiPo battery
